# Run Web 命令

在 Chrome 浏览器中运行 Flutter Web。

## 步骤

1. **启动 Web**
   ```bash
   bash scripts/run-web.sh
   ```

2. **验证**
   - 浏览器打开 http://localhost:PORT/
   - 检查控制台无报错

## 输出格式

```markdown
## Flutter Web 启动状态

- ✅ 启动成功
- 🌐 地址: http://localhost:PORT/
```
